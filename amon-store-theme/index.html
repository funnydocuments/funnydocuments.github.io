Template Features

Amonstore - Magento 2 Theme được xây dựng bám theo cấu trúc của magento core và kế thừa từ theme blank. Với hàng trăm cấu hình, giúp bạn dễ tùy biến để có một giao diện ưng ý mà không làm hỏng bất kỳ tính năng nào của magento core.
Bạn có thể dử dụng Amonstore - Magento 2 Theme để xay dựng bất cứ site nào bạn muốn như supermarket store, digital store, hi-tech store, watch store, fashion store, clothing store, furniture store, book store, cosmetics shop, luxury jewelry & accessories store or mega stores.
Ngoài tính năng của magento, Amonstore Theme còn các thêm các tính năng nổi bật dưới đây.

  1. Không giới hạn số lượng mầu: hỗ trợ nhiều định dạng mầu bao gồm HEX, RGB, RGBA...
  2. Nhiều layout cho header, content và footer.
  3. Hỗ trợ không giới hạn số lượng google font
  4. Sticky Menu
  5. Header mobile thân thiện và dễ sử dụng
  6. Sản phẩm có nhiều cấu hình giúp bạn có thể dễ dàng tùy biến để có nhiều kiểu dáng khác nhau
  7. Hiển thị countdown ở phần mô tả sản phẩm
  8. Hiển thị recently view slider ở trang sản phẩm với nhiều cấu hình cho slider
  9. “New” and “Sale” Product Labels
  10. Lazy Load Images: Images will be loaded quickly when scrolling down!
  11. Support Cloud Zoom and Pop-up images on the detail page
  12. 100% table-less design, based on CSS3 with many animated effects
  13. Popup: Nội dung popup linh hoạt, bạn có thể sử dụng như một newsletter popup hoặc có thể đặt một nội dung bất kỳ
  14. Custom Css/Js: Hỗ trợ upload css/js file
  15. Category: Hỗ trợ từ 1 đến 8 sản phẩm trên một dòng
  16. Widget CMS Block Slider: Hiển thị slider một cách dễ dàng như slidershow, testimonial slider...
  17. Widget Filter Product Slider: Hiển thị sản phẩm một cách dễ dàng vào bất cứ nơi nào bạn muốn
  18. Module Import/Export Content: Giúp bạn backup/restore dữ liệu cms page, cms block cùng với media, đồng thời sử dụng để cấu hình giống demo một cách dễ dàng
  19. Module Ajax Layer Navigation: Giúp bạn lọc sản phẩm một cách nhanh chóng
  20. Module Quickview: giúp bạn xem thông tin sản phẩm một cách nhanh chóng
  21. Module Custom Tab: Hiển thị custom tab theo attribute của sản phẩm, giúp bạn dễ dạng tạo thêm thông tin cho một nhóm sản phẩm có thuộc tính bất kỳ
  22. Import/Export configuration
  23. Right to Left Language

  • System Requirements – Required system credentials for your magento installation.
  • Installation Guide – Installation guide that describes how to install magento2.
  • Online Magento 2 guide – Online magento documentation that describes about magento2 in detail.
  • Magento Forum – Magento forum site.
  • stackoverflow.com – You can also get some useful articles from stackoverflow.

How To Install

There are two ways to install a Magento Theme:

  1. Quickstart Installation: Suitable if you have your first installation and want to get a whole new fresh site as our Demo.
  2. Manual Installation (including Theme Installation): Suitable if your store is full with content that you don't want to overwrite.

Quickstart Install

Manual Install

In this guide, We assume that you already have Magento installed, so to prepare installation, you need to do the following steps:

  1. We recommend you to duplicate your live store on a staging/test site and try installation on it in advance
  2. Backup magento files and the store database.
    It’s very important to backup all of themes and extensions in Magento before installation, especially when you are working on a live server. We strongly recommend you to do not omit this step.
  3. Disable all cache on your Magento site include:
    • Magento Cache (System > Cache Management)
    • Any additional cache on your server, for instance PHP cache engines, APC, ...
  4. Enable Maintenance mode for your store before uploading theme source.
  5. Download theme package amonstore-themeonly-mX.X.X_vX.X.X.zip (zip file), extract this package and upload folders: app, pub to your Magento folder. You can use a FTP software, such as FileZilla, then logging into your hosting
  6. Make sure that you upload all theme folders and files successfully

Fonts
Main Font

Font Family: Font chữ cho website. Hỗ trợ các font cơ bản và google font

Google Font Link: Liên kết tới Google Font

Google Font Family: Tên Google Font bạn muốn sử dụng

  1. Vào trang: https://fonts.google.com/
  2. Chọn hoặc tìm font bạn muốn sử dụng: ví dụ font Heebo
  3. Click vào dấu + để chọn font
  4. Tại tab CUSTOMIZE: chọn kích thước font mà bạn muốn sử dụng. Chúng tôi khuyến cáo bạn sử dụng dạng font Regular(400) và Bold(700)

    View Image

  5. Quay lại tab EMBED: Nhập link và font family vào hai input tương ứng là Google Font Link và Google Font Family

    View Image

Google Font Weight: Nhập: 400(Regular). Những text in đậm sẽ tự nhận Font Weight: 700(Bold)

Elements Google Font

Mục này để hiển thị google font cho một thành phần xác định được định danh thông qua class hoặc ID. Cách nhập google font giống phần Main Font.

Google Font Elements: Nhập class hoặc ID của các element bạn muốn hiện thị Google Font

Tip: Bạn chỉ nên chọn Google Font có thời gian load nhanh(Load time) để website của bạn được tối ưu nhất.
